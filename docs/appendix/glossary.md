# Glossary of Terms

## A

### Affordance
In ecological psychology, an affordance is a property of the environment that provides an opportunity for action to an organism. In robotics, it refers to the potential actions that an object or environment offers to a robot.

### Action Space
The set of all possible actions that an agent can perform in an environment. In robotics, this includes all possible motor commands.

### Actuator
A component of a machine responsible for moving and controlling a mechanism or system. Common types include servo motors, stepper motors, and hydraulic cylinders.

### Affordance Detection
The process by which a robot identifies the possible interactions it can have with objects in its environment.

### Artificial Neural Network (ANN)
Computational models inspired by biological neural networks that consist of interconnected groups of nodes that process information using connectionist approaches.

## B

### Bayesian Inference
A method of statistical inference in which Bayes' theorem is used to update the probability for a hypothesis as more evidence or information becomes available.

### Behavior Tree
A mathematical model of plan execution used in robotics and game development to describe the hierarchical structure of tasks.

### Bipedal Locomotion
The motion pattern of walking on two legs, a key challenge in humanoid robotics.

### Biomimetic
Design that模仿s or takes inspiration from biological systems and elements found in nature.

## C

### Central Pattern Generator (CPG)
A neural network capable of producing rhythmic patterns of motor activity without sensory feedback or rhythmic inputs.

### Compliance Control
Control strategy that allows a robot to adapt its behavior based on contact forces with the environment.

### Convolutional Neural Network (CNN)
A class of deep neural networks commonly applied to analyzing visual imagery.

### Control Lyapunov Function
A mathematical function used to prove the stability of control systems.

### Cyber-Physical System
A system of collaborating computational elements controlling physical entities.

## D

### Deep Learning
A subset of machine learning based on artificial neural networks with representation learning.

### Deep Reinforcement Learning
Combination of deep learning and reinforcement learning where neural networks are used as function approximators.

### Degrees of Freedom (DOF)
The number of independent parameters that define the configuration of a mechanical system.

### Differential Drive
A common configuration of wheeled robots using two wheels mounted parallel to each other and driven independently.

### Dynamic Movement Primitive (DMP)
A method for learning and reproducing motor skills based on dynamical systems theory.

## E

### Embodied Cognition
The theory that cognitive processes are deeply rooted in the body's interactions with the environment.

### Embodiment
The physical form of an intelligent agent and its interaction with the environment.

### End Effector
The device at the end of a robot arm that interacts with the environment (e.g., gripper, tool).

### Episodic Memory
Memory system that records experiences that a person has had, important for lifelong learning in robots.

## F

### Forward Kinematics
The use of joint angles to compute the position and orientation of the end effector.

### Force Control
Control strategy focused on regulating contact forces between the robot and environment.

### Fused Lasso
A regression method that performs variable selection and regularization simultaneously.

## G

### Gaussian Process
A stochastic process where any finite set of variables has a joint Gaussian distribution, used for probabilistic modeling.

### Generalized Coordinates
A set of coordinates that uniquely defines the configuration of a system, taking constraints into account.

### Geometric Control
Application of differential geometry to control problems in robotics.

### Grit
A measure of perseverance and passion for long-term goals, important for researchers in robotics.

## H

### Hierarchical Reinforcement Learning
Approach to RL that decomposes complex tasks into simpler subtasks arranged in a hierarchy.

### Humanoid Robot
A robot with a body structure similar to that of a human, typically having a head, torso, two arms, and two legs.

### Hybrid Zero Dynamics
A control approach for underactuated robotic systems that combines continuous and discrete dynamics.

## I

### Inverse Kinematics
The process of determining the joint angles required to achieve a desired end-effector position and orientation.

### Impedance Control
A control strategy that regulates the dynamic relationship between position and force.

### Intrinsic Motivation
Motivation that comes from within an individual rather than from external rewards, important for autonomous learning.

### Iterative Learning Control
A control scheme that improves performance by learning from repeated executions of the same task.

## J

### Jacobian Matrix
Matrix of all first-order partial derivatives of a vector-valued function, used in robotics to relate joint velocities to end-effector velocities.

### Jumping Conductor
A type of flexible electronic component used in soft robotics and wearable devices.

## K

### Kalman Filter
An algorithm that uses a series of measurements observed over time to estimate unknown variables.

### Kinematics
The study of motion without considering the forces that cause it.

### Kinodynamic Planning
Motion planning that considers both kinematic and dynamic constraints.

## L

### Latent Space
An abstract multi-dimensional space representing hidden features in data, used in generative models.

### Learning from Demonstration
A technique where robots learn tasks by observing human demonstrations.

### Linear Quadratic Regulator (LQR)
A control technique that computes optimal control inputs by minimizing a quadratic cost function.

### Locomotion
The ability to move from one place to another, a key capability in legged robots.

## M

### Manipulability Ellipsoid
A geometric representation of a robot's ability to apply forces and velocities in different directions.

### Model Predictive Control (MPC)
An advanced control method that uses a model of the system to predict future behavior and optimize control actions.

### Morphological Computation
The idea that part of the computational burden traditionally assigned to the controller can be offloaded to the physical body.

### Motor Babbling
Random motor exploration used by infants and robots to learn about their bodies and environment.

## N

### Natural Actor-Critic
A reinforcement learning algorithm that uses natural gradients for policy improvement.

### Neurorobotics
An interdisciplinary field combining neuroscience and robotics to understand brain function.

### Newton-Euler Algorithm
Recursive algorithm for computing inverse dynamics of serial-link manipulators.

## O

### Operational Space Control
A control framework that allows specifying task-space behavior while maintaining null-space properties.

### Omnidirectional Wheels
Wheels that can move in any direction, useful for mobile robot platforms.

### Open-Loop Control
Control without feedback, where the control action is not dependent on the output.

## P

### Passive Dynamic Walking
Walking behavior that emerges from the mechanical design and environmental interaction without active control.

### Path Planning
The process of determining a route from a start to a goal position.

### Physical Intelligence
Intelligence that emerges from the interaction between an agent's physical form and its environment.

### Policy Gradient
A class of reinforcement learning algorithms that directly optimize the policy function.

### Probabilistic Robotics
Approach to robotics that explicitly accounts for uncertainty in sensing and actuation.

## Q

### Q-Learning
A model-free reinforcement learning algorithm to learn quality values for state-action pairs.

### Quaternion
Mathematical construct used to represent rotations in 3D space without singularities.

## R

### Reactive System
A system that continuously adapts its behavior based on environmental changes.

### Reinforcement Learning
A type of machine learning where agents learn to make decisions by interacting with an environment.

### Riemannian Metric
A metric tensor that defines distances and angles in curved spaces, used in geometric mechanics.

### Robot Operating System (ROS)
Flexible framework for writing robot software that provides services for hardware abstraction and message passing.

### ROS 2
The next generation of Robot Operating System with improved real-time capabilities and security.

### RRT (Rapidly-exploring Random Tree)
A motion planning algorithm that builds a space-filling tree to find paths in high-dimensional spaces.

## S

### Sensor Fusion
The process of combining data from multiple sensors to improve accuracy and reliability.

### Simultaneous Localization and Mapping (SLAM)
The computational problem of constructing or updating a map of an unknown environment while simultaneously keeping track of an agent's location.

### Singularity
A configuration of a robot where it loses one or more degrees of freedom.

### Soft Robotics
Branch of robotics that deals with robots made from flexible materials that can safely interact with humans.

### State Estimation
The process of estimating the internal state of a system from noisy measurements.

### Supervised Learning
Machine learning approach where the algorithm learns from labeled training data.

## T

### Task Space
The space in which the robot's task is naturally expressed (e.g., Cartesian space for end-effector position).

### Trajectory Optimization
Mathematical approach to finding optimal paths by formulating the problem as an optimization.

### Trust Region Method
Optimization technique that restricts the step size to ensure convergence.

## U

### Underactuated System
A mechanical system with fewer actuators than degrees of freedom.

### Unified Robot Description Format (URDF)
XML format for representing robot models in ROS.

### Upper Confidence Bound (UCB)
Algorithm for balancing exploration and exploitation in multi-armed bandit problems.

## V

### Variational Inference
Statistical method for approximating probability densities through optimization.

### Visual Servoing
Control strategy that uses visual feedback to control robot motion.

### Volumetric Methods
Techniques that represent 3D space as a discretized volume for planning and mapping.

## W

### Whole-Body Control
Control approach that considers the entire robot dynamics when computing control actions.

### Wheeled Inverted Pendulum
A robot model that balances on two wheels like a Segway.

### Workspace
The set of all positions that the end effector of a robot can reach.

## X, Y, Z

### Zero-Moment Point (ZMP)
A criterion for static and dynamic stability of legged robots.

### Ziegler-Nichols Method
Empirical method for tuning PID controllers.
