# Hardware Requirements

## Minimum System Requirements

To work through this textbook and implement the projects, you'll need the following minimum hardware:

### Development Computer
- **CPU**: Quad-core processor (Intel i5 or AMD Ryzen 5 equivalent)
- **RAM**: 16 GB DDR4
- **Storage**: 500 GB SSD
- **GPU**: Dedicated GPU with 4GB VRAM (NVIDIA GTX 1060 or equivalent)
- **OS**: Ubuntu 20.04 LTS or 22.04 LTS (preferred), Windows 10/11 with WSL2, or macOS 12+

### Recommended System
- **CPU**: Hexa-core or more (Intel i7/i9 or AMD Ryzen 7/9)
- **RAM**: 32 GB DDR4
- **Storage**: 1 TB NVMe SSD
- **GPU**: NVIDIA RTX 3070 or better (for accelerated ML training)
- **Network**: Gigabit Ethernet, 802.11ac Wi-Fi

## Robot Platforms

### Entry Level
- **TurtleBot 3 Burger/Waffle**: Ideal for learning ROS 2 basics
- **Robotis OP3**: Humanoid platform for locomotion experiments
- **Locobot**: Mobile manipulator for perception-action tasks

### Mid Range
- **Unitree Go1/A1**: Quadruped robots for dynamic locomotion
- **Boston Dynamics Spot**: Advanced quadruped for research
- **ANYmal**: Quadruped robot for outdoor navigation

### Advanced
- **Unitree H1**: Full-sized humanoid robot
- **Agility Robotics Digit**: Humanoid for logistics applications
- **Tesla Optimus**: Experimental humanoid platform

## Sensor Systems

### Cameras
- **Stereo Cameras**: Intel RealSense D435/D455, ZED Stereo Camera
- **RGB-D Cameras**: Kinect v2, Intel RealSense D435i
- **Event Cameras**: Prophesee, Insightness for high-speed vision

### LiDAR
- **2D LiDAR**: Hokuyo UTM-30LX, Sick TIM571
- **3D LiDAR**: Velodyne VLP-16, Ouster OS1, Livox Horizon/Mid-40

### IMU Sensors
- **Inertial Measurement Units**: VectorNav VN-100, XSens MTi-G-700
- **IMU with GPS**: Advanced navigation systems

### Force/Torque Sensors
- **6-axis F/T Sensors**: ATI Gamma, Nano17 series
- **Tactile Sensors**: GelSight, FingerVision for manipulation

## Computing Platforms

### Edge Computing
- **NVIDIA Jetson Series**: 
  - Jetson Xavier NX (recommended for humanoid robots)
  - Jetson Orin AGX (high-performance option)
  - Jetson Nano (entry-level option)
- **Intel RealSense**: T265 for tracking, D400 series for depth
- **Google Coral**: Edge TPU for AI inference

### High-Performance Computing
- **NVIDIA GPUs**: RTX 3080/3090, RTX 4080/4090 for training
- **Cloud Platforms**: AWS EC2 p3/p4 instances, Google Cloud TPUs

## Network Infrastructure

### Local Network
- **Switch**: Managed gigabit switch for robot networking
- **Access Point**: Dual-band Wi-Fi 6 for reliable connectivity
- **Cables**: Cat6e cables for wired connections

### Communication Protocols
- **Ethernet**: For high-bandwidth sensor data
- **Wi-Fi**: For mobile robot connectivity
- **5G**: For remote operation of robots

## Workshop Equipment

### Tools
- **Multimeter**: Digital multimeter for electrical measurements
- **Oscilloscope**: For signal analysis (optional)
- **Soldering Iron**: Temperature controlled for electronics assembly
- **3D Printer**: For custom robot parts (Prusa MK3S+, Ender 3)

### Safety Equipment
- **Safety Glasses**: For workshop activities
- **First Aid Kit**: Basic emergency supplies
- **Fire Extinguisher**: Class C for electrical fires

## Software Licenses

### Commercial Software
- **MATLAB/Simulink**: For control system design
- **SolidWorks**: For mechanical design (student license available)
- **ADAMS**: For mechanical simulation (optional)

### Simulation Platforms
- **Gazebo**: Open-source simulation (included with ROS)

- **NVIDIA Isaac Sim**: For high-fidelity simulation

## Budget Considerations

### Academic Lab Setup (10 stations)
- **Low-cost option**: $50,000 - $100,000
- **Mid-range option**: $100,000 - $250,000
- **High-end option**: $250,000 - $500,000

### Individual Student Setup
- **Minimum**: $2,000 - $3,000 (simulations only)
- **Recommended**: $5,000 - $8,000 (with entry-level robot)
- **Advanced**: $10,000+ (with mid-range robot platform)
