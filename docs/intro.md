# Introduction to Physical AI & Humanoid Robotics

Welcome to the comprehensive textbook on **Physical AI & Humanoid Robotics**! This 13-week course covers the cutting-edge intersection of artificial intelligence and robotics, focusing on embodied intelligence and autonomous systems.

## Course Overview

This textbook is organized into four comprehensive modules that progressively build your understanding of Physical AI and humanoid robotics:

### Module 1: The Robotic Nervous System (ROS 2) - Weeks 1-5
- **Weeks 1-2**: Introduction to Physical AI and embodied intelligence
- **Weeks 3-5**: ROS 2 Fundamentals
  - Nodes, Topics, Services, Actions
  - URDF (Unified Robot Description Format)
  - rclpy Python integration
  - Sensor systems (LIDAR, cameras, IMUs, force sensors)

### Module 2: The Digital Twin (Gazebo & Unity) - Weeks 6-7
- **Week 6**: Robot Simulation with Gazebo
- **Week 7**: Unity for high-fidelity rendering
  - Physics simulation and sensor modeling
  - URDF/SDF format conversions
  - Collision detection and environment building

### Module 3: The AI-Robot Brain (NVIDIA Isaac) - Weeks 8-10
- **Week 8**: NVIDIA Isaac SDK and Isaac Sim introduction
- **Week 9**: AI-powered perception and manipulation
- **Week 10**: Reinforcement learning and sim-to-real transfer
  - Isaac ROS integration
  - VSLAM (Visual Simultaneous Localization and Mapping)
  - Nav2 path planning for bipedal movement
  - Jetson Orin integration

### Module 4: Vision-Language-Action (VLA) - Weeks 11-13
- **Weeks 11-12**: Humanoid Robot Development
  - Kinematics and dynamics
  - Bipedal locomotion and balance control
  - Manipulation and grasping
- **Week 13**: Conversational Robotics and Capstone Project
  - Voice-to-Action with OpenAI Whisper
  - LLM cognitive planning
  - Autonomous humanoid with voice commands

## Interactive Learning Experience

This textbook features an integrated **Qwen-powered chatbot** that can answer questions about the content, explain complex concepts, and provide additional insights. Simply select any text and ask the chatbot to elaborate on the topic.

## Hardware Focus

Throughout this course, we'll explore industry-standard platforms:

- **NVIDIA Isaac** ecosystem for AI-powered robotics
- **ROS 2** (Robot Operating System) for robotic software development
- **Jetson Orin** for edge computing in robotics
- **Unitree** and other humanoid platforms
- **Gazebo** and **Unity** for simulation

## Prerequisites

Before starting this course, you should have:

- Intermediate programming experience in Python
- Basic understanding of linear algebra and calculus
- Familiarity with Linux command line
- Basic knowledge of robotics concepts (helpful but not required)

## Learning Approach

This textbook takes a hands-on approach to learning Physical AI and humanoid robotics. Each week includes:

- Theoretical foundations
- Practical examples and code implementations
- Simulation exercises
- Real-world applications
- Capstone project components

## Getting Started

Navigate to the first module to begin your journey into Physical AI and humanoid robotics. Each module builds upon previous concepts, so we recommend following the sequence from Week 1 to Week 13.
